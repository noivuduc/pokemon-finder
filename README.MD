## A Pokemon Finder Application

## Application type
- Terminal application

## Environments Requirements
- Node.js version >= 10

## How to run
### cd to the application folder
### Install the packages
```bash
   npm install
```
### Give the application permission to execute (on Mac OS)
```bash
    chmod +x ./pokemon 
```
### Run the application
- Mac OS
```bash
    ./pokemon
```
 - Window OS
```bash
    node pokemon
```
